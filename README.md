# Todo App - Next.js + Spring Boot

A modern Todo Dashboard with **Next.js 15 (App Router)** frontend and **Spring Boot 3** backend.

## Tech Stack

### Frontend
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Server Components + Client Components

### Backend
- Spring Boot 3
- Spring Data JPA
- MySQL
- Lombok
